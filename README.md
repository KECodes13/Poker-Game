# Texas Hold'em Hand Analyzer

A web-based Texas Hold'em poker hand strength analyzer built with Spring Boot. This application deals random poker hands and evaluates hand strength in real-time.

## Features

- **Random Hand Generation**: Automatically deals 2 hole cards and 5 community cards
- **Hand Strength Analysis**: Identifies poker hand rankings from High Card to Royal Flush
- **Clean Web Interface**: Simple HTML/CSS frontend for easy interaction
- **RESTful API**: Spring Boot backend with endpoints for hand generation and analysis

## Hand Rankings Detected

The analyzer recognizes all standard poker hands:
- Royal Flush
- Straight Flush
- Four of a Kind
- Full House
- Flush
- Straight
- Three of a Kind
- Two Pair
- One Pair
- High Card

## Technologies Used

- **Backend**: Java, Spring Boot
- **Frontend**: HTML, CSS
